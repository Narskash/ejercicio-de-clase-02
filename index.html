Actualizar repositorio remoto con cambios locales

Primeros cambios en repositorio local

Añadiendo archivos nuevos

git add "file"

git add .

Confirmación del/los cambio/s

Empaquetamos todos los cambios antes de subirlos al repositorio remoto

git commit -m "titulo descriptivo del commit"

Subiendo los cambios al repositorio remoto

git push origin main

Tip

add y commit a la vez

git commit -am "titulo descriptivo del commit"
